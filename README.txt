This project is a review of the article Bilateral counterparty risk valuation with stochastic dynamical models and application to CDS; Damiano Brigo and Agostino Capponi; Sept 23, 2008

We present a complete summary of the article as well as few critics in the file "Credit_Derivatives_BRCVA.pdf".
The original article is available here: https://arxiv.org/pdf/0812.3705.pdf

In the Python notebook "BRCVA" you will find an implementation of the algorithms proposed by the authors.
